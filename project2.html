Our Idea

DIY Clinometer for measuring height and distance.

Get ready to be amazed as you apply mathematical concepts to real-world measurements. You'll be surprised at how easy it is to calculate heights and distances using simple tools and a bit of trigonometry.

Materials Required

  • Straw: Provides the base structure for the clinometer, acting as a sighting tube.
  • Thread: Used to create a plumb line, which ensures accurate vertical measurements.
  • Weight: Attaches to the end of the thread, keeping it hanging and vertical.
  • Glue: Secures the thread and other components to the straw, creating a sturdy clinometer.
  • Cardboard: Used to create a protractor, which helps measure angles.

Costing

  • Straw and Glue: ₹5
  • Thread and Weight: around free
  • Cardboard: around free
  • Total: ₹5

Process

  • Prepare the base: Create a protractor base on the cardboard.
  • Fix the straw: Tape a straw horizontally along the handmade protractor.
  • Attach the plumb line: Tie a weight at the end of a thread and fix it at the midpoint of the protractor horizontal.
  • Testing: Point the straw at the top of an object and measure its distance or height using trigonometry.

Implications

  • Educational Tool: This DIY clinometer can serve as a hands-on learning tool in schools.
  • Practical Applications: Much useful in engineering and architecture.
  • Cost Effective and Eco-Friendly: By using waste material, we can create a very helpful tool.

Future Scope

  • Integrate Technology: Preciser measurements by integrating technology.
  • Comparative Studies: Conduct experiments comparing the accuracy of DIY products and commercialised ones.
